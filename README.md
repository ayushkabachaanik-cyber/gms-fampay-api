## 🌟 What is GMS FamPay API?

<img align="right" src="https://user-images.githubusercontent.com/74038190/235294012-0a55e343-37ad-4b0f-924f-c8431d9d2483.gif" width="200"/>

> **GMS FamPay API** is a **free, open-source UPI payment gateway** — just like Razorpay, but 100% free forever. Accept FamPay UPI payments in your **websites, Telegram bots, Discord bots, and mobile apps** with just one API call.

```
  No fees. No limits. No middleman. Just pure UPI payments.
  Works with FamPay @fam UPI IDs — scan → pay → done.
```

## 🚀 Quick Start (3 Steps)

### Step 1 — Sign Up

Visit the live site → **Sign Up** tab:
- Enter your Gmail address
- Enter your **FamPay UPI ID** (open FamPay → Profile → UPI ID, e.g. `yourname.12345@fam`)
- Enter your Google App Password

### Step 2 — Get API Key

Your API key appears instantly in the dashboard. Copy it.

### Step 3 — Make Payments 🎉

```bash
# Generate a payment QR
GET https://yoursite.com/api/qr?api=YOUR_API_KEY&amount=100

# Verify payment status
GET https://yoursite.com/api/verify?api_key=YOUR_API_KEY&order_id=FAMPAY...
```

## 📡 API Reference

<div align="center">

| Method | Endpoint | Description | Auth |
|:------:|----------|-------------|:----:|
| `GET` | `/api/qr?api=KEY&amount=100` | Generate UPI QR + order ID | API Key |
| `GET` | `/api/verify?api_key=KEY&order_id=ID` | Check payment status | API Key |
| `GET` | `/qr/ORDER_ID.png` | Raw QR image (PNG) | None |
| `POST` | `/webhook/payment` | Confirm payment instantly | Webhook Secret |
| `POST` | `/auth/signup` | Create account | None |
| `POST` | `/auth/login` | Login → get JWT | None |
| `POST` | `/auth/update-upi` | Update FamPay UPI ID | JWT |
| `POST` | `/auth/revoke-api` | Regenerate API key | JWT |

</div>

## 💳 Payment Status Flow

```
  Customer scans QR (FamPay / any UPI app)
              │
              ▼
         [PENDING] ──── 5 min timer ───────────► [EXPIRED ❌]
              │
              │  webhook POST /webhook/payment
              │  OR manual confirm from backend
              ▼
         [SUCCESS ✅]
              │
              └──► polling frontend updates in ~4 seconds
```

## 🤖 Telegram Bot Example (Python)

```python
import requests, time

API_KEY = "YOUR_GMS_API_KEY"
BASE    = "https://your-replit-url.replit.dev"

def create_payment(chat_id, amount, bot):
    r    = requests.get(f"{BASE}/api/qr?api={API_KEY}&amount={amount}")
    data = r.json()["data"]
    bot.send_photo(chat_id, data["qr_url"],
                   caption=f"💳 Pay ₹{amount} via FamPay\nOrder: `{data['order_id']}`\n⏳ Expires in 5 min")
    return data["order_id"]

def wait_for_payment(order_id, timeout=300):
    for _ in range(timeout // 5):
        r      = requests.get(f"{BASE}/api/verify?api_key={API_KEY}&order_id={order_id}")
        status = r.json()["status"]
        if status == "success": return True
        if status == "expired":  return False
        time.sleep(5)
    return False
```

## 🌐 Discord Bot Example (Node.js)

```javascript
const fetch = require('node-fetch');
const API_KEY = 'YOUR_GMS_API_KEY';
const BASE    = 'https://your-replit-url.replit.dev';

async function sendPaymentQR(interaction, amount) {
  const res  = await fetch(`${BASE}/api/qr?api=${API_KEY}&amount=${amount}`);
  const data = (await res.json()).data;

  await interaction.reply({
    content: `💳 **Pay ₹${amount} via FamPay**\nOrder ID: \`${data.order_id}\`\n⏳ Expires in 5 minutes`,
    files: [{ attachment: data.qr_url, name: 'payment.png' }]
  });

  // Poll for payment
  const poll = setInterval(async () => {
    const v = await fetch(`${BASE}/api/verify?api_key=${API_KEY}&order_id=${data.order_id}`);
    const s = (await v.json()).status;
    if (s === 'success') {
      interaction.followUp('✅ Payment received! Thank you.');
      clearInterval(poll);
    } else if (s === 'expired') {
      interaction.followUp('❌ Payment expired. Please try again.');
      clearInterval(poll);
    }
  }, 5000);
}
```

## 🌍 Website Embed Example (HTML)

```html
<!DOCTYPE html>
<html>
<body>

<button onclick="generatePayment()">Pay ₹99</button>
<div id="qr-container"></div>

<script>
const API_KEY = 'YOUR_GMS_API_KEY';
const BASE    = 'https://your-replit-url.replit.dev';

async function generatePayment() {
  const res  = await fetch(`${BASE}/api/qr?api=${API_KEY}&amount=99`);
  const data = (await res.json()).data;

  document.getElementById('qr-container').innerHTML =
    `<img src="${data.qr_url}" width="250"/><p>Order: ${data.order_id}</p>`;

  // Auto-check every 4 seconds
  const poll = setInterval(async () => {
    const v = await fetch(`${BASE}/api/verify?api_key=${API_KEY}&order_id=${data.order_id}`);
    const s = (await v.json()).status;
    if (s === 'success') { alert('✅ Payment done!'); clearInterval(poll); }
    if (s === 'expired') { alert('❌ Expired.'); clearInterval(poll); }
  }, 4000);
}
</script>
</body>
</html>
```
